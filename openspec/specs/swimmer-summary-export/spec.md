# swimmer-summary-export Specification

## Purpose
Define the swimmer summary export template used to present standout
performances, grade status, milestone progress, and recent improvement in a
shareable format.

## Requirements

### Requirement: Swimmer summary export SHALL present standout highlights
The system SHALL provide a swimmer summary export that highlights the swimmer's
most representative events, recent progress, and milestone context in a format
optimized for reading and sharing.

#### Scenario: Administrator opens a swimmer summary export
- **WHEN** an administrator opens the swimmer summary export
- **THEN** the page presents strongest event highlights, goal status, and
  recent improvement summaries instead of only raw performance rows
