## ADDED Requirements

### Requirement: Admin swimmer management SHALL remain usable on mobile
The admin swimmer management page SHALL render a mobile-friendly member list on
phone-sized viewports instead of relying only on the desktop table.

#### Scenario: Administrator opens swimmer management on mobile
- **WHEN** the admin swimmer page is viewed on a phone-sized viewport
- **THEN** existing swimmers render in a mobile-readable list that stays within
  the viewport width
