# 公开档案整页长图与竞技场聚焦化 Requirements

## Overview

当前公开个人档案页上的“保存分享海报”导出结果为空白，已经不适合作为继续迭代的基础能力；同时竞技场页的“赛道切换”独立卡片在信息结构上有些多余，用户更需要一个聚焦后的赛道详情主视图，并让排行榜前三得到更明确的视觉强调。

## Problem Frame

### A. 公开个人档案页的分享导出不可用

当前个人档案页虽然提供了“保存分享海报”，但导出结果在实际使用中已经出现空白。对用户来说，这意味着这个按钮不是“体验不好”，而是“功能失效”。

与其继续保留一个语义模糊、结果不稳定的“海报保存”按钮，更合理的方向是把它明确升级为：
- 对当前档案页做整页长图导出
- 把用户正在看的整页成长档案，从头到尾导出成一张长图

### B. 竞技场的信息重心仍然不够集中

当前竞技场页已经有“赛道详情”作为主分析区，但“赛道切换”仍然单独占用一个块级区域。它虽然承担了切换功能，但从用户视角看，会形成“左边一个切换区，右边一个详情区”的双栏注意力分裂。

用户更可能想要的是：
- 直接进入“赛道详情”主视图
- 需要切换赛道时，在详情内部完成，不再额外单独占一整块区域
- 把页面注意力集中到一个最重要的内容区域

### C. 排行榜前三不够突出

当前赛道详情的排名列表虽然已经完整，但前三名和其他名次之间的视觉层级差异还不够明显。既然竞技场的核心就是竞争，前三名应该得到更明确但克制的强调。

## Users

- 家长：希望一键把孩子整页成长档案保存成长图，而不是得到一个空白导出。
- 教练：希望竞技场主要看详情，不被多余的结构切走注意力。
- 学员：希望在竞技场里一眼看到谁是前三、谁更突出。

## Goals

- 移除当前无效的“保存分享海报”语义，改为稳定可用的“整页长图导出”。
- 让公开个人档案页支持从头到尾导出当前成长页内容。
- 收掉竞技场里独立的“赛道切换”块，把切换能力融入赛道详情主视图。
- 让赛道详情中的前三名有明确、统一、融入主题的视觉强调。
- 保持当前系统主色调、Radix/shadcn 组件语义和移动端兼容性。

## Non-Goals

- 不引入浏览器原生或系统级截图权限。
- 不做社交平台分享 SDK。
- 不重做竞技场后端分组协议。
- 不做夸张动画或与当前风格脱节的竞技特效。

## Key Product Decisions

### 1. 用“整页长图导出”替换“保存分享海报”

用户看到的按钮和行为都应该切换成新的语义，例如：
- 下载整页长图
- 保存成长长图

关键不是文案，而是功能含义要明确：导出的对象是“整个公开档案页”，不是一个局部海报卡片。

### 2. 整页导出不等于浏览器截图 API

浏览器页面自身无法直接调用系统级“整页截屏”能力，因此实现上应采用“整页导出容器”的方式，把当前页的完整成长内容渲染成一个稳定的长图源，再生成图片下载。

这在产品上等价于“整页长图”，比继续保留当前失效的局部导出更可靠。

### 3. 竞技场改成单主视图

推荐方向：
- 删除独立的“赛道切换”卡片
- 保留赛道切换能力，但把它收进“赛道详情”卡片内部
- 赛道详情成为页面唯一核心主区块

这样页面结构会从：
- 左边切换，右边详情

变成：
- 一个聚焦的赛道详情主区块，内部附带可切换的赛道概览

### 4. 前三名强化要有层级，但不能做成浮夸竞技特效

推荐强化方式：
- Top 1 使用最强视觉强调，例如冠冕/高光/呼吸感
- Top 2 / Top 3 使用次一级强调，例如银/铜徽标和轻微动效
- 所有强化必须建立在当前主题色与卡片语言之上

## Recommended Direction

推荐做成 **“公开档案整页长图导出 + 竞技场详情聚焦化 + 前三名层级强化”**。

原因：
- 它直接修复了当前失效功能。
- 它让公开档案页和竞技场页都更聚焦，而不是继续叠加次要结构。
- 它保留了用户需要的切换能力，但不再把切换本身做成页面的第二主角。

## Alternative Considered

### A. 保留“保存分享海报”，只继续微调截图参数

不推荐。

因为这会继续沿用一个已经被用户明确感知为失效的功能语义。

### B. 直接删除导出功能，不补新能力

不推荐。

因为用户明确还需要导出，只是需要的是“整页长图”，不是空白海报。

### C. 保留赛道切换卡片，只给前三名加特效

不推荐。

因为这只解决了“层级不够明显”，没有解决竞技场结构注意力分散的问题。

## Success Criteria

- 个人档案页不再出现“保存分享海报”这个失效功能。
- 用户可以从个人档案页导出完整成长长图。
- 竞技场页面不再单独出现“赛道切换”卡片。
- 赛道详情内部仍能切换分组，并保持清晰易用。
- 排行榜前三的视觉强调明显，但整体风格仍与当前主题一致。
